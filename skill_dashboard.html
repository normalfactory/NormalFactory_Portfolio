Lead the team in building the CrimeView Dashboard application that is a multi-tenant application that provides a geospatial view of public safety data. The application is deployed on-premise of some of the largest agencies in the country. Additionally, it is also deployed within AWS Gov on VMs with over 150 clients.

One of the local clients is the San Diego Harbor Police.

Technology
  • C#
  • SQL Server
  • Windows Communication Foundation (WCF)
  • ArcGIS API for Silverlight
  • ArcGIS Server
  • SignalR

LINQ

One of the goals of the application was to have a highly interactive user experience. The application was written in Silverlight and this framework included LINQ. When the user clicks map legend or charts, the counts in all of the charts are updated as well as the map.

I was responsible for the creation of the core mapping component that had this filtering capability build with LINQ.

ArcGIS Server

A custom Server Object Extension (SOE) for ArcGIS Server, written in C# and ArcObjects, was used to do the geoprocessing. The initial release used DCOM to communicate from the website to the SOE and this was later migrated to Restful calls.

Business Intelligence

Beyond the CrimeView Dashboard, I have used off the shelf Business Intelligence (BI) tools. The first BI tool was Crystal Reports where a desktop product was integrated to export data into a database so that a report template could be pointed to the dataset.

I have also worked with SiSense, PowerBI and Tableau.
