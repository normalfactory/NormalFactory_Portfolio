Teams

I have worked in agile development environments that were high performing. Beyond great team members, it also requires organization. Outlined below are a number of items that contributed to the success of the team.

Project Structure

All of the Visual Studio solutions had a common structure that allow developers to quickly find items and on-board new developers. It also ensures that best practices are implemented.

Class Diagrams

The class diagrams within Visual Studio are used for each namespace of the Data Transfer Objects (DTO). This allows way to visualize the hierarchy between the objects.

Clean Code

One part of high quality code is comments. All of the DTO have XML comments to provide information to the developer on the purpose.

Wiki

Atlassian Confluence was used to document functionality beyond the comments in the code. One example was the schema of the databases were in Confluence.
