Strategy

I have the ability to work with stakeholders on product roadmaps and then create the architecture required to achieve that roadmap. This was done at TriTech where I was responsible for creating a strategy for the mapping technology used by the products.

TriTech has a number of products that use mapping and all use different technologies. This resulted in wasted development efforts and difficulty with the deployments. I worked with a large collection of stakeholders; from upper management, product management and the support team; to get their feedback and requirements. Using the my experience of GIS technology, I was able to create a technology roadmap.

This roadmap is to have a core set of mapping components that could be used by the different applications. It includes an embedded map that is cross-platform; works on Windows, iOS and Android with the use of Xamarin. Additionally, it also included a JavaScript browser map.

This plan is going to take years to implement. The first part of that roadmap with the embedded map has been completed and it has resulted in improved performance, a better user experience and lower licensing costs.
